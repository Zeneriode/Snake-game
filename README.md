# Игра змейка
Это игровой проект.

Первый проект на Python, который возможно будет сделан даже не очень кривым.

---

***ИНСТРУКЦИЯ***

Чтобы протестировать (поиграть) в наш проект надо:

* установить python 3.10 с официального сайта. 
* клонировать проект. 
* загрузить библиотеку pygame 2.1.2.
